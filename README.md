# Kişisel Portfolyo Web Sitesi

Bu proje, **HTML, CSS ve JavaScript** kullanılarak geliştirilmiş kişisel portfolyo web sitemdir.  
Projelerimi, yeteneklerimi ve iletişim bilgilerimi sade ve modern bir tasarımla sunmayı amaçlamaktadır.

## Özellikler

- Responsive (duyarlı) tasarım
- Menü linkleri için smooth scroll (yumuşak kaydırma)
- Sayfa kaydırıldığında stil değiştiren sabit navbar
- Modern hero alanı
- Projeler ve iletişim bölümü

## Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript (Vanilla JS)
